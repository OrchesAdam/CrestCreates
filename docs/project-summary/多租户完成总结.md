## 🎯 核心特性

### 1. 灵活的租户识别

支持 5 种识别策略，可单独使用或组合：

```csharp
// 单一策略
ResolutionStrategy = TenantResolutionStrategy.Header

// 组合策略（优先级从左到右）
ResolutionStrategy = TenantResolutionStrategy.Header | 
                     TenantResolutionStrategy.Subdomain
```

### 2. 三种隔离策略

| 策略 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| **Database** | 大客户、高安全 | 完全隔离、易备份 | 资源消耗大 |
| **Discriminator** | 小客户、成本优先 | 资源高效、易管理 | 隔离度低 |
| **Schema** | 中等规模 | 平衡性能和隔离 | 管理复杂度中等 |

### 3. 自动数据过滤

```csharp
// 无需手动过滤，自动应用租户过滤器
var products = await _dbContext.Products.ToListAsync();
// ↑ 仅返回当前租户的数据

// 插入时自动设置 TenantId
_dbContext.Products.Add(new Product { Name = "Product 1" });
await _dbContext.SaveChangesAsync();
// ↑ TenantId 自动填充
```

### 4. 安全防护

```csharp
// 防止跨租户修改
var product = await _dbContext.Products.FindAsync(id);
product.Name = "Updated";
await _dbContext.SaveChangesAsync();
// ↑ 如果 product.TenantId != currentTenant.Id，抛出异常
```

### 5. 灵活切换租户

```csharp
// 临时切换租户上下文
using (_currentTenant.Change("tenant2"))
{
    // 此作用域内访问 tenant2 的数据
    var data = await _dbContext.Products.ToListAsync();
}
// 作用域结束后自动恢复原租户
```

## 💡 使用示例

### 快速开始（3 步配置）

```csharp
// 1. 注册服务
services.AddMultiTenancyWithInMemory(
    options => options.ResolutionStrategy = TenantResolutionStrategy.Header,
    provider => {
        provider.AddTenant(new TenantInfo("t1", "Tenant 1", "ConnStr1"));
        provider.AddTenant(new TenantInfo("t2", "Tenant 2", "ConnStr2"));
    });

// 2. 添加中间件
app.UseMultiTenancy();

// 3. 使用
public class ProductController : ControllerBase
{
    private readonly ICurrentTenant _currentTenant;
    
    [HttpGet]
    public IActionResult GetInfo()
    {
        return Ok($"Current Tenant: {_currentTenant.Id}");
    }
}
```

### 鉴别器模式（单数据库多租户）

```csharp
// 1. 定义实体
public class Product : MultiTenantEntity<Guid>
{
    public string Name { get; set; }
}

// 2. 配置 DbContext
protected override void OnModelCreating(ModelBuilder modelBuilder)
{
    modelBuilder.ConfigureTenantDiscriminator(_currentTenant);
}

// 3. 自动工作
var products = await _dbContext.Products.ToListAsync();
// ↑ 仅查询当前租户数据，无需手动过滤
```
